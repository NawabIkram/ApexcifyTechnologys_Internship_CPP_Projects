# 🏦 Bank Management System (C++)

## 📌 Description
A console-based Bank Management System developed in C++ that simulates real-world banking operations. This project allows users to create accounts, perform transactions, and manage customer data efficiently using file handling.

---

## ✨ Features
- 🆕 Create New Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 🔍 Search Account by ID
- 📄 Display All Accounts
- 🗑️ Delete Account
- 💾 Persistent Data Storage (File Handling)

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures (if used)
