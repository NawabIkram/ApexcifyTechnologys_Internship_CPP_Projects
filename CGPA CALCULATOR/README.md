# 🎓 CGPA Calculator (C++)

## 📌 Description
A simple C++ program that calculates CGPA based on user input for subjects and grades. It demonstrates basic logic building and user interaction.

---

## ✨ Features
- Input subject grades
- Calculate SGPA/CGPA
- Simple and efficient logic
- User-friendly console interface

---

## 🛠️ Technologies Used
- C++
- Basic Programming Concepts

---

## 🧠 Concepts Used
- Loops
- Conditional Statements
- Functions
- Arithmetic Calculations

---

## ▶️ How to Run
g++ cgpa_calculator.cpp -o cgpa
./cgpa
