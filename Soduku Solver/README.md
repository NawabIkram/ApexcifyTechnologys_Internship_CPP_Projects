# 🧩 Sudoku Solver (C++)

## 📌 Description
A C++ program that solves a Sudoku puzzle using the backtracking algorithm. This project demonstrates recursion and problem-solving techniques.

---

## ✨ Features
- Solves standard 9x9 Sudoku
- Uses backtracking algorithm
- Displays solved board
- Efficient recursive solution

---

## 🛠️ Technologies Used
- C++
- Recursion
- Algorithms

---

## 🧠 Concepts Used
- Backtracking
- Recursion
- 2D Arrays
- Constraint Checking

---

## ▶️ How to Run 
g++ sudoku.cpp -o sudoku


./sudoku
