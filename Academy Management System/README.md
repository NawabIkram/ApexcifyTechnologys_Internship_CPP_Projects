# 🎓 Academy Management System (C++)

## 📌 Description
A console-based application to manage student records, courses, and academic data.

---

## ✨ Features
- Add / Delete Students
- Manage Courses
- File Handling for Data Storage
- Menu-driven Interface

---

## 🛠️ Technologies
- C++
- OOP Concepts
- File Handling

## ▶️ How to Run

g++ main.cpp -o academy
./academy
